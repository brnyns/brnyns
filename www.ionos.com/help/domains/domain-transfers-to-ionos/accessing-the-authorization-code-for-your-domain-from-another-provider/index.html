Accessing the Authorization Code for Your Domain from Another Provider

Please use the “Print” function at the bottom of the page to create a PDF.

This article will show you how to access the authorization code from various providers/registrars.

If you wish to transfer your domain to IONOS, you will need this authorization code to approve (authorize) the domain transfer on IONOS's end.

Select Your Provider

To display the appropriate instructions, choose the provider with which you want to access the authorization code and prepare the domain transfer:

  • GoDaddy
  • Network Solutions
  • Bluehost
  • eNom
  • HostGator

Note: You can obtain the authorization code for your domain transfer from your current provider at any time, regardless of the remaining term of your domain.

Is your domain registered with another provider?

If your domain is registered with another provider, you can find important information about preparing for your domain transfer in the article Prerequisites for a Successful Domain Transfer.

You can find detailed assistance and instructions on accessing your authorization code from the customer support or documentation provided by your current provider.
