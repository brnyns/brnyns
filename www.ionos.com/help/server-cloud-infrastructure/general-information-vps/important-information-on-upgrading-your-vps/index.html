Important information on upgrading your VPS

Please use the “Print” function at the bottom of the page to create a PDF.

For Linux VPS and Windows VPS managed in the Cloud Panel

This articles guides you through upgrading your VPS.

The following server resources can be increased with an upgrade:

  • CPU
  • RAM
  • Disk space

During the upgrade, the server will be unavailable for a few minutes.

Notes

  • With VPS contracts, you can only upgrade to higher versions.
  • If you have reached the maximum capacity of a Linux VPS server, you can switch to a higher Linux package.
  • If you have reached the maximum capacity of a Windows VPS Windows server, you can switch to a higher Windows package.
  • It is not possible to downgrade a VPS. If you wish to have a server with less resources, your only option is to manually migrate your data to a new, smaller server contract.
  • When you upgrade your server, the storage space on the hard disk is also increased. There is no automatic resizing of partitions within Linux or Windows. To utilize the additional storage space, you must first increase the partition within the operating system (Linux or Windows). All data remains intact after the upgrade. No data is expected to be lost, however, creating a backup first is always good practice.
  • Virtual Server Cloud contracts are excluded from the ability to upgrade. These contracts can be identified by their name, "Virtual Server Cloud". If you require more resources than is available on your current Virtual Server Cloud, your only option is to manually migrate your data to a new, larger server contract.

To upgrade your VPS, or to find out which options are available to you, complete the following steps:

Requirement

The server must be turned on before the upgrade.

  • Log in to your IONOS account and go to the Contracts & subscriptions page. You will see an overview of your contracts and the additional services you have ordered.
  • Click on the name of the contract you want to change. This will display the contract details.
  • In the Contract change tile at the bottom, click on  Display contract change options.
  • You will now be redirected to the IONOS customer shop to see an overview of the available upgrades. To continue with the upgrade, or to find out more about a larger package, select the desired size.
  • To authorize the contract upgrade, click on the Switch to this plan button in the tile with the desired package.
  • Read and accept the General Terms and Conditions.
  • Click on Buy now.

    The VPS contract change will then be carried out automatically. You will receive all the important information about your upgrade by email.
