Supported Cloud Backup Features by Operating System

Please use the “Print” function at the bottom of the page to create a PDF.

The Cloud Backup features are supported by the following operating systems:


Microsoft Windows

  • Windows 7 Service Pack 1 and newer

  • Windows Server 2008 R2 Service Pack 1 (Windows Defender Antivirus Management is supported by Windows 8.1 and newer.)

  • Windows Server 2012/2012 R2 - all editions

  • Windows Server 2016, Windows Server 2019 and Windows Server 2022 - all installation options (x64 only), with the exception of Nano Server

  • Windows 10 - the Home, Pro, Education and Enterprise editions (x64 only)

  • Windows 11 - all editions


Linux

  • CentOS 6.10, CentOS 7, CentOS 8*

  • CentOS Stream 8*, CentOS Stream 9*

  • Debian 12 and older

  • Ubuntu 24.04 and older

  • AlmaLinux 9.4* and older*

  • Rocky Linux 9.4* and older*

  • CloudLinux 5.x, 6.x, 7.x, 8.x*, 9.4*

  • Red Hat Enterprise Linux 8.x*, 9.0*, 9.1*, 9.2*, 9.3*, 9.4* and older

  • Fedora 40 and older

  • Oracle Linux 8.x*, 9.0*, 9.1*, 9.2*, 9.3*, 9.4* and older

  • SUSE Linux Enterprise Server 15

*From version 8.4, only supported with kernels from 4.18 to 5.19


macOS

10.13.x and newer (only antivirus and anti-malware protection, and device control are supported). Device Control features are supported on macOS 10.15 and newer, or macOS 11.2.3 and newer.

For more detailed information on the supported operating systems, please refer to the following page:

https://dl.managed-protection.com/u/baas/help/24.12/user/en-US/index.html#supported-operating-systems-and-environments.html 

Please note

The operating systems Windows 7 and Windows Server 2008 R2, CentOS 6.10, CentOS 8 and CloudLinux 6.10 are no longer supported by the manufacturer. Furthermore, Windows 8.1 has not received any security updates or technical support from Microsoft since 10 January 2023.

Please note that IONOS customer service does not provide technical support for operating systems that are no longer supported by the manufacturers.
